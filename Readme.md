# LifeGame

# How to Play

## Github Pages

[LifeGame Github Pages](https://shin-sakata.github.io/LifeGame/)

## Local

```
$ git clone https://github.com/shin-sakata/LifeGame.git
$ cd LifeGame
$ npm install
$ npx elm make src/Main.elm --optimize
Success!     

    Main ───> index.html
```

Open index.html in browser
